# 👗 Dress Reservation System

A desktop application developed in **C#** using **WPF** for managing dress reservations and rental operations.

## Features

- 👗 Dress inventory management
- 📅 Reservation scheduling
- 👤 Customer management
- 📦 Rental and return tracking
- 🔍 Search and filter functionality
- 💾 Local database integration
- 🖥️ User-friendly WPF interface

## Technologies

- C#
- WPF (.NET)
- XAML
- SQLite
- Entity Framework *(if used)*
